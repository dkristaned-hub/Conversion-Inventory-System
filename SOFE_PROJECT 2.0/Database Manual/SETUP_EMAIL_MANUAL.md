# Simple Email Setup

## Set Up Gmail for the System

### Step 1: Get Gmail Ready
1. Go to your Gmail account
2. Click on your profile picture (top right)
3. Click "Manage your Google Account"
4. Go to "Security" on the left
5. Turn on "2-Step Verification" if not already on

### Step 2: Create App Password
1. In the same Security section, find "App passwords"
2. You might need to sign in again
3. Select "Mail" from the app dropdown
4. Select your device (or "Other")
5. Enter a name like "Inventory System"
6. Click "Generate"
7. Copy the 16-character password that appears

### Step 3: Tell the System About Your Gmail
1. Find the file called `email_functions.php`
2. Open it with a text editor
3. Look for the Gmail settings section
4. Replace the example email with your Gmail address
5. Replace the example password with the app password you copied

### Step 4: Add Your Email to the System
1. Login to the system as admin
2. Go to "Profile Settings"
3. Add your email address where it asks for it

That's it! The system will now send you email alerts when important things happen.
