# Simple Guide to Conversion Inventory System

This system helps you manage your cooking oil business. It tracks employees, inventory, invoices, and more.

## What It Does

- **Employee Management**: Add staff, track attendance, and manage accounts
- **Inventory Control**: Keep track of oil stock levels and monitor changes
- **Invoice Tracking**: Create and manage customer invoices
- **Activity Logs**: See what happens in the system
- **User Accounts**: Different access levels for admin and staff

## Quick Setup

### Step 1: Install XAMPP
- Download XAMPP from: https://www.apachefriends.org/
- Install it on your computer
- This gives you everything needed to run the system

### Step 2: Start the Services
- Open XAMPP
- Click "Start" for Apache
- Click "Start" for MySQL
- You should see green lights

### Step 3: Put Files in Place
- Copy this system folder to: `C:\xampp\htdocs\`
- Rename the folder to something simple like "inventory"

### Step 4: Set Up Database
- Open browser and go to: `http://localhost/inventory/database_setup.php`
- The system will create the database automatically

### Step 5: Login
- Go to: `http://localhost/inventory/Login.html`
- Username: `admin123`
- Password: `admin123`

## First Time Use

1. Login with the default account
2. Go to Profile Settings and change the password
3. Add your employees using the Employee tab
4. Start adding your inventory items

## Help

If something doesn't work:
- Make sure XAMPP is running (green lights)
- Check that files are in the right folder
- Try refreshing the page

This system saves all your data safely in a database.
